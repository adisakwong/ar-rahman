# myQuran : Recitation 🕋

A premium, mobile-responsive web application designed for the recitation of the Holy Quran. This app features a clean, traditional layout inspired by the Mushaf with modern web aesthetics.

## ✨ Features

- **Surah Selection**: Easily switch between **Surah Ar-Rahman** and **Surah Al-Mulk**.
- **Recitation-Focused Design**: Large, clear Arabic typography (Amiri font) optimized for mobile reading and focus.
- **Traditional Layout**: Verse numbers are displayed in Arabic digits within ornamental symbols (﴿ ﴾) at the end of each verse, flowing naturally like a real Mushaf.
- **Premium Aesthetics**: Glassmorphism effects, minimalist spacing, and a custom-generated Islamic geometric background.
- **Smart CORS Handling**: Built-in logic to handle API cross-origin restrictions, ensuring reliability across different hosting environments.

## 🚀 Live Demo
You can view the live app by enabling **GitHub Pages** for this repository.

## 🛠 Tech Stack

- **HTML5** & **CSS3** (Backdrop filters, Animations, Mobile-first)
- **Vanilla JavaScript** (ES6+, Async/Await)
- **QuranAPI**: Data sourced from [quranapi.pages.dev](https://quranapi.pages.dev/)
- **Google Fonts**: Amiri, Sarabun

## 🌐 Deployment to GitHub Pages

1. Push this code to a new repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Branch**, select `main` and `/ (root)`.
4. Click **Save**.
5. Your app will be live at `https://<your-username>.github.io/<repo-name>/`.
