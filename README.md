# myQuran : Recitation 🕋

A premium, mobile-responsive web application designed for the recitation of the Holy Quran. This app features a clean, traditional layout inspired by the Mushaf with modern web aesthetics.

## ✨ Features

- **Surah Selection**: Easily switch between available Surahs through the persistent top navigation bar.
- **Dynamic Settings**: Personalize your reading experience via the integrated settings panel:
    - **Font Size**: Adjustable Arabic typography size (1.5rem - 4.5rem) for optimal legibility.
    - **Number Style**: Toggle between Traditional Arabic digits (๑, ๒, ๓) and Western digits (1, 2, 3).
    - **Display Mode**: Full **Dark Mode** and Light Mode support to reduce eye strain.
- **Recitation-Focused Design**: Large, clear Arabic typography (Amiri font) optimized for mobile reading and focus.
- **Traditional Layout**: Verse numbers flow naturally like a real Mushaf, positioned at the end of each Ayat.
- **Premium Aesthetics**: Glassmorphism effects, minimalist spacing, and a subtle Islamic geometric background.
- **Persistence**: All user preferences (size, theme, etc.) are saved automatically to your browser.
- **Reliable Data**: Smart CORS handling ensures consistent data fetching from the API.

## 🚀 Live Demo
You can view the live app by enabling **GitHub Pages** for this repository.

## 🛠 Tech Stack

- **HTML5 & CSS3**: Custom CSS variables, Backdrop filters, Smooth Animations, and Responsive Design.
- **Vanilla JavaScript**: ES6+, Async/Await, State Management, and LocalStorage persistence.
- **QuranAPI**: Data sourced from [quranapi.pages.dev](https://quranapi.pages.dev/)
- **Typography**: Amiri (Quranic Arabic), Noto Naskh Arabic, and Sarabun (Thai/English UI).

## 🌐 Deployment to GitHub Pages

1. Push this code to a new repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Branch**, select `main` and `/ (root)`.
4. Click **Save**.
5. Your app will be live at `https://<your-username>.github.io/<repo-name>/`.
